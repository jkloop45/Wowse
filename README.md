# 介绍 #
Wows! 是某 Wows 的辅助工具.因为我看了一圈这类的都没有发现什么好的工具(*妈盒子), 于是打算自己写一个.
目前服务器使用的是 LeanCloud , 如有更好的欢迎推荐.

## 独立账号 ##
用于发布插件或提交评论.

## 插件发布 ##
支持 游戏版本号, 插件版本号, 自动更新, 用户反馈, 用户评论.

## 插件管理 ##
分类管理: 音效插件, UI插件, 舰船涂装, 游戏内容.

## 插件评论 ##
支持赞, 回复 等功能.

## 战绩查询 ##
通过 [rank.kongzhong.com](rank.kongzhong.com) 来查询战绩(侵删).

## 独立插件格式包 ##
兼容wsp(多玩盒子的插件)格式.

具体格式:
- 第 1 byte: 插件类型版本号.
- 第 2-5 bytes: 插件作者.
- 第 5-20 bytes: 插件描述.
- 第 20-25 bytes: 插件版本号.
- 第 25-30 bytes: 支持的游戏版本号.
- 第 30-40 bytes: 插件id.
- 第 40+ bytes: 插件内容.

## 语言支持 ##
支持 简体中文, 繁体中文, 英语 等语言.
